.container{
  display:grid;
  grid-template-columns:1fr 1fr 1fr;
  grid-template-rows:1fr 1fr 1fr;
  height:50vh;
  width:400px;
}


--->grid-template-columns:1fr 1fr 1fr;(  Create three boxes in each column having having equal size )

--->grid-template-rows:1fr 1fr 1fr;(  Create three boxes in each rows having equal size )


--->height:50vh;  vh means Viewpoint height   -->The full form of VH is viewport 
height. It works like the percentage unit as well.
 Specifying 10vh is equivalent to occupying 10% of entire visible screen height.


--->width:400px;  it means it takes upto 400px horizontal length to create the boxes.
